Mobile First

Antigamente, os sites eram feitos pensados para serem visualizados apenas no monitor de um computador desktop ou notebook. Porém, com os smartphones, surgiu a necessidade de adaptar a web para os dispositivos móveis. O conceito de Mobile First surgiu para se referir à criação de qualquer projeto web que prioriza a experiência em celulares, enquanto são feitas adaptações para desktop e outras plataformas.

Media Queries

Media queries são basicamente blocos de código em CSS que são executados quando a resolução de tela atual atinge certo tamanho determinado. Esse é o recurso mais usado para adaptar conteúdos dentro de um site e deixá-lo responsivo.

Observe no menu de navegação acima os media queries sendo usados, para deixar os links na vertical e as cores de fundo que ficam mais fortes conforme a resolução aumenta e mais fracas quando diminui. Dentro das medias queries, propriedades já estabelecidas no CSS por seletores podem ser sobrescritas.
